# zcrack by zabry0

Cyberpunk hacker-aesthetic PWA for iPhone.

## Features

- 💀 Matrix rain + animated skull canvas terminal
- 🌍 IP Geo Locator — multi-source lookup with automatic fallback (4 APIs, private-IP detection, no single dependency)
- 📡 Wi-Fi Sniffer — live fake network scanner with signal bars
- 💻 Remote Access Terminal — SSH-flavoured interactive terminal (15+ commands)
- 🎙 Sound Sniffer — real-time Web Audio API FFT spectrum analyser

## Install as PWA (iPhone)

1. Open the URL in Safari
2. Tap **Share → Add to Home Screen**
3. Launch from home screen for full-screen experience
